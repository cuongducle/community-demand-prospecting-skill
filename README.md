# Community Demand Prospecting Skill

`community-demand-prospecting` is a reusable skill for turning an unfinished product or repository into a marketing-ready brief, researching the market and user landscape, choosing a positioning angle, and drafting safe community outreach for channels such as Reddit and X.

## What It Covers

- repo-to-product-brief audit
- launch-readiness review
- market, user, competitor, and substitute research
- positioning-angle selection
- buyer-intent query generation
- thread scoring and reply drafting
- guardrails against spammy outreach

## Repository Layout

- `community-demand-prospecting/`: the actual skill bundle
- `community-demand-prospecting/SKILL.md`: main workflow
- `community-demand-prospecting/references/`: focused references loaded as needed
- `community-demand-prospecting/assets/`: reusable templates

## Install

Copy the `community-demand-prospecting/` folder into your local skills directory, or install it from this repository using whatever skill workflow your agent runtime supports.

## Entry Point

Start with [community-demand-prospecting/SKILL.md](./community-demand-prospecting/SKILL.md).
